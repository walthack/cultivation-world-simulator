| Contract | Proof | Status |
|---|---|---|
| Fill on vs off → mechanical projection byte-identical | `test_scenario_narrative_fill_boundary.py::test_enabling_fill_does_not_perturb_mechanical_state` (cross-tick differential) | ✅ |
| narration never read by AI-memory path | `..._boundary.py::test_ai_roleplay_prompt_context_reads_content_not_narration`; `test_event_storage.py::test_narration_*absent/stripped*` (DB SELECT + in-memory render) | ✅ |

**Deferred (honest):** no Playwright API+UI narration-display contract was added in this pass. A live-scenario Playwright narration render is a post-merge tooling item, not a merge blocker.
